## üìå Project Overview
This project presents an exploratory sales analysis of a coffee shop dataset using **Microsoft Excel**.  
The goal is to extract business insights from transactional data through **data cleaning, pivot tables, KPIs, and visualizations**.

The analysis focuses on understanding **sales performance over time**, **customer behavior**, and **product performance**.

---

## üóÇ Dataset
- Source: Kaggle (Coffee Sales Dataset)
- Files:
  - `index_1.csv`
  - `index_2.csv`
- Records include:
  - Date and hour of transaction
  - Product sold
  - Payment method (cash / card)
  - Final transaction amount

---

## üßπ Data Cleaning
Data preparation was performed directly in Excel and includes:
- Converting date and time fields to proper formats
- Standardizing numeric values (amounts)
- Handling empty or inconsistent values
- Creating helper columns for analysis (hour extraction)

Cleaned data is stored in structured Excel tables to support pivot analysis.

## üìä KPIs
The following general KPIs were calculated:

- **Number of Transactions**

These KPIs provide a high-level overview of overall business performance.

## üß† Key Insights
- Higher daily revenue is generally associated with higher transaction volume.
- Peak hours concentrate both higher sales volume and higher revenue.
- A small group of core products accounts for most transactions and revenue.
- Premium products contribute meaningfully through higher average prices, even with lower sales volume.
- Revenue generation is balanced between core and premium products rather than dominated by only one group.
